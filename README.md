# Engineering-core-Theory-and-code
A comprehensive repository mapping theoretical engineering concepts from Mechanics, Mathematics and BEEE into structured Python scripts, supplemented by verified, handwritten exam paper solutions to bridge academic theory with practical application.

## 🛠 Subject Breakdown

### 📁 Mechanics: Lesson 01- Resultant of Forces
- 'Analytical_solver.py': An engineering calculator designed to resolve multiple concurrent coplanar forces into horizontal and vertical components to find the precise magnitude and direction of the absolute resultant force.
- 'graphical_solver.py' : A visualisation script utilising Matlibplot to plot concurrent coplanar vectors originating from a central point, mapping the final resultant force on a   2D coordinate plane.
